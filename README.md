# SpaceShip_LUA_L-VE2D_Midterm
Player is a spaceship, meteors at collision destroy player, player can shoot lasers which when collided with meteor destroys meteor and destroys the laser. 10 score for each second survived + 50 score per meteor destroyed. art for sprite background is directly done by or inspired from work of Tsukumizu's japanese comic (manga) Shimeji Simulation
